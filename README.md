# PhotoSelector
A local mediaselector for Android(一款本地图片/视频/音频选择器)

目前使用sketch作为本库的图片加载器-[sketch](https://github.com/panpf/sketch)
目前6.0以上系统还没有加入动态申请权限功能，可以自己添加该功能
目前有的功能可以自行下载demo，首页有详细配置
还有一些功能未完成，后续会加入
